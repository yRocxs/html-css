Computador entende ondas quadradas, famoso codigo binario

1024 Bytes = 1 KB
1024 Kb = 1 MB
1024 Mb = 1 GB
1024 Gb = 1 TB
1024 Tb = 1 Petabyte
1024 Pb = 1 Exabyte
1024 Eb = 1 Zetabyte
1024 Zb = 1 Yotabyte

Existe difenrença em MB para Mb? Sim.
MB = Megabytes = Armazamento
Mb = Megabits = Transmissão

Para nos conectamos na Internet e preciso duas coisas:

Um computador (Ou outro dispositivo) que sera o Cliente e a Internet que é passada para um modem ou conetada por telefone.
Entretanto o telefone ou tvs apresentam uma frequencia Ondulada e o computador um frenquencia quadrada, neste caso o Modem faz o trabalho de Modulação e Demodulação.


Como acessamos servidores?

Utilizamos o IP do cliente para se conectar no IP do servidor do site que queremos entrar.

Porem para não precisarmos saber o IP de cada site, ele usa um sistema de DNS para salvar esse IP com outro nome, como uma Agenda de Celular.

Apos receber digitar o link do site, ele é enviado para o DNS e o DNS te retorna o IP atualizado do site, assim permitindo a conexão.
