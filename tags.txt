<h1>Titulo(Cabeçalho)</h1>

Podemos usar o h1, h2, h3, h4, h5 e h6. Para titulos e sobre-titulos.

<p>Paragrafo</p>

<hr> Criar linha (Linha Horizontal)

<p>Voce pode escrever um parágrafo de qualquer jeito. colocar tudo no meio do par de tags <p> e </p></p>

<p>Se precisar quebrar o texto em algum lugar específico <br> como esse, voce pode usar a tag br.</p>

Para utilizar os simbolos em HTML usamos: & (E comercial) e a tag.

        &reg;
        &copy; 
        &trade;
        &euro;
        &pound;
        &yen;
        &cent;
        &Delta;
        &uarr;

Para utilizar emojis em HTML usamos: &#x e o codigo obtido no Emojipedia.org
        &#x1F596; 
        &#x1F913;

Para adicionar imagens usamos a tag: 

<img src=" x " alt=" x ">

(Aonde contem o X botamos a imagem ou o texto)

Para usar imagens flexiveis para diversos dispositivos usamos a tag:

<picture>
        <source media="(max-width: 750px)" srcset="imagens/foto-p.png" type="image/png">
        <source media="(max-width: 1050px)" srcset="imagens/foto-m.png" type="image/png">
        <img src="imagens/foto-g.png" alt="Imagem flexível">
</picture>

OBS: Sempre colocamos no IMG a imagem principal para o site, e os outros tamanhos imagens iguais, porem em resolução menores.
Um site sem imagem flexivel diminui a taxa que é entregada para o publico.

Note que a tag <source> possui três atributos:
‣ type vai indicar o media type da imagem que usamos (veja mais informações
sobre media types no capítulo 10)
‣ srcset vai configurar o nome da imagem que será carregada quando o tamanho
indicado for atingido
‣ media indica o tamanho máximo a ser considerado para carregar a imagem
indicada no atributo srcset. 

Para adicionar Favicons usamos a esta tag no Head:

<link rel="shortcut icon" href=" x " type="image/x-icon">

(Aonde contem o X botamos a imagem)

<strong> </strong> deixa o termo escolhido em Negrito

<em> </em> deixa o termo escolho em Itálico

<mark> </mark> Marca uma parte do texto.

<big> </big> <small> </small> Deixa o texto grande ou pequeno.

<ins> </ins> Texto inserido para dar uma enfase e indicar que ele foi adicionadom depois

<sup> </sup> Elevar textos

<sub> </sub> Colocar texto em baixo
